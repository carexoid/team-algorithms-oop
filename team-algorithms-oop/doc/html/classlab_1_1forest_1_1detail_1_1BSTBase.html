Detailed Description

template<typename T, typename Compare, typename DerivedTree>
class lab::forest::detail::BSTBase< T, Compare, DerivedTree >

Abstract base class for binary search tree implementations using CRTP.

Provides BST iterator, representing elements in increasing order and begin(), end() methods. Provides search and size methods, copy and move constuctors, copy and move assigment operators, and swap function. Requires insertImpl(const T& key) and eraseImpl(const T& key) methods in derived class. Implementation details like root and size are protected.
